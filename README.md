## Содержание

Исследование посвящено построению всепогодного портфеля на индексах Московской биржи с использованием многокритериальной оптимизации (Шарп, Сортино, Кальмар, CVaR ratio, Martin ratio, Omega) и эконометрического анализа факторной структуры доходности.

Основные результаты:

- Снижение максимальной просадки портфеля с 23–24% до 15–16% относительно базовой 1/N-аллокации
- Рост коэффициента Кальмара с 0,77 до 1,09
- Эконометрическая модель связи доходности портфеля с макрофакторами (MOEXBMI, EUR/RUB, оборот организаций, импорт) с R² = 0,60

Полный текст диссертации доступен по запросу автору.

## Требования

- Python 3.10+
- Основные библиотеки: `pandas`, `numpy`, `scipy`, `statsmodels`, `matplotlib`, `cvxpy`
- Jupyter Notebook или JupyterLab

Установка зависимостей:

```bash
pip install pandas numpy scipy statsmodels matplotlib cvxpy jupyter
```

## Запуск

```bash
jupyter notebook asset_selection.ipynb
```

Ноутбук выполняется последовательно сверху вниз. Все пути к данным относительные и привязаны к папке `data/`.

## Источники данных

Исторические ряды индексов Московской биржи получены через ISS API (https://iss.moex.com) за период ВСТАВЬ_СВОЙ_ПЕРИОД. Используемые тикеры:

- ВСТАВЬ_СВОИ_ТИКЕРЫ (например: MCFTR, RGBITR, RUCBTRNS, RUGOLD)

Макроэкономические переменные — Росстат и Банк России.
